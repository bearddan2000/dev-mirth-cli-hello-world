# dev-mirth-cli-hello-world

## Description
A POC for mirth programming language.

## Tech stack
- g++
- make

## Docker stack
- ubuntu:22.04
- rzuckerm/mirth

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
